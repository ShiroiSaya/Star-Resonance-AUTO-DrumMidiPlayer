# Star-Resonance-AUTO-DrumMidiPlayer

星痕共鸣架子鼓自动演奏工具，兼容绝大多数 MIDI 鼓谱文件。将 MIDI 鼓谱实时转换为键盘按键，专为节奏游戏和鼓类 MIDI 播放设计。

## 📦 下载

### 直接下载（推荐）

前往 [Releases](https://github.com/ShiroiSaya/Star-Resonance-AUTO-DrumMidiPlayer/releases) 页面下载最新版本的 `SayaDrumPlayerV1.0.exe` 或其他版本。

**当前最新版本**：SayaDrumPlayerV1.0

**系统要求**：
- Windows 7/8/10/11
- 需要以管理员身份运行

### 从源码运行

```bash
# 克隆仓库
git clone https://github.com/ShiroiSaya/Star-Resonance-AUTO-DrumMidiPlayer.git

# 安装依赖
pip install mido python-rtmidi pydirectinput pygetwindow

# 运行程序
python Dr_MidiPlayer_BySayaTech.py
